To select the word where your cursor is pointed, the most common and efficient methods use either the mouse or specific keyboard shortcuts. Using the Mouse The standard method across most operating systems (Windows, macOS, Linux) and applications is: Double-click the word. This will select the entire word the cursor is currently inside or adjacent to. You can then click and drag to extend the selection to adjacent words or characters, if needed. Windows & Linux The most reliable sequence that works in many applications: Press Ctrl + Left Arrow to jump to the beginning of the current or previous word. Then, press Ctrl + Shift + Right Arrow to select the word to the right (which is now the current word). Alternatively, some specific applications support a single shortcut: In some code editors like Sublime Text and Atom, Ctrl + D selects the word under the cursor. Then, Option + Shift + Right Arrow to select it.
